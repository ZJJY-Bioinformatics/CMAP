What's new

Different Barplot Module Optimization [2023-07-03]

We have added a species-level parameter to the module, allowing users to selectively display desired species levels.

Composition Module Optimization [2023-06-15]

Large samples in the species composition module required excessive computational time. We have redesigned the species composition module to now only require a single computation. Subsequent modifications to graphic settings no longer necessitate recomputation.

New PSM Module Added [2023-05-10]

We've just added the Propensity Score Matching (PSM) module to enhance data analysis capabilities by reducing selection bias and confounding: 16S -->> Data Processing -->> PSM.

Homepage design [2023-04-27]

Our homepage has been re-designed to ensure increased compliance with accessibility guidelines..

Added Metagenomics analysis modules [2023-04-07]

We are adding the metagenomics analysis module, where the species analysis module is similar to the existing 16S analysis module, and the gene function analysis module will be available soon.
